Comandos 

git add <lista de arquivos>

 - Adiciona os arquivos novos e novos e modificados para o próximo commit
	git add . 

git commit [-m "Message]

	- Registra o commit com tosdos os arquicos que usou "git add"

git status
git log 
gi add
git commit 
git push
git diff <commit1> <commit2> (compara as alterações dos commit's)
git reset <commit> (Resetar o repositório para um determinado commit)
git clone <URL>
git checkout <commit> (volta para versões anteriores)
git checkout -- . (Desfaz todas as alterações antes do ultímo commit)

git branch
git branch <nova_branch>
git branch -d <branch> - deleta uma branch
git checkout <branch> - muda de branch
git merge <branch> - aplica commit's de uma branch em uma branch atual
Comandos importantes no terminal:
